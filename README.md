# Chrome Extension with WebPack

## Start

Create Manifest File

    {
        "manifest_version": 3,
        "name": "ReactJS Boilerplate",
        "description": "Meu primeiro test",
        "version": "1.0.0"
    }

Import Image: 16 48 128

    "icons": {
            "16": "icon.png",
            "48": "icon.png",
            "128": "icon.png"
        }

Init NPM

    npm init -y
